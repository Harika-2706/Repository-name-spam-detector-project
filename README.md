# 📩 Spam Detection App

This is a machine learning-based spam detection system built using Python and Flask.

## 🚀 Features
- Detect spam messages using ML model
- Show confidence score
- Report spam messages
- View reported spam history

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- HTML/CSS

## ▶️ How to Run
1. Install dependencies:
   pip install flask pandas scikit-learn

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
